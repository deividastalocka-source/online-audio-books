# online-audio-books
My university project featuring an e-commerce website for online audio books
Note: This is a front-end only project. Payments, backend systems, and real user accounts are not included.

Features
Browse a collection of audiobooks
Individual book detail pages
Basic cart interface (UI only)
Responsive layout
Built with HTML, CSS, and JavaScript

How to Run the Project Locally
Download or clone the repository.
Open the project folder.
Open index.html in your web browser.
